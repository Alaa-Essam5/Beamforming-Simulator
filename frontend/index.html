Beamforming Guide
• N↑: HPBW ∝ 1/N, sharper beam, more sidelobes
• Spacing >0.5λ: grating lobes appear
• Freq↑: λ↓ → denser near-field fringes
• Steer↑: HPBW broadens as 1/cos(θ)
• Amp↑: brighter field, SNR up
• Apodization: trades HPBW for lower SLL
• SNR↓: noise floor rises, SLL masked

Mode Guide
A-mode: Echo amplitude vs depth. Peaks = impedance boundaries. Higher frequency = sharper spikes but less depth. Attenuation weakens echoes with depth; TGC compensates.

B-mode: 2D sonogram built from many A-lines. Move probe or click "Full Sweep". Lateral blur = focal depth effect. Brighter = stronger echo.

Doppler — KEY RULE:
Δf = 2·f₀·v·cos(θ)/c
θ = beam direction vs flow direction
θ=90° always → Δf=0 (⊥ = no shift)
θ=0° → max shift (∥ = full signal)
+Δf = flow toward probe (red on chart)
-Δf = flow away from probe (blue)
Use A/D to tilt beam, watch θ change
